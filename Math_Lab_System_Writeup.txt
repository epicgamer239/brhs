BRHS MATH LAB SYSTEM
====================

A Digital Tutoring Platform for Broad Run High School

OVERVIEW
--------
The BRHS Math Lab System is a user-friendly online platform designed to connect students who need help with mathematics to qualified tutors at Broad Run High School. Built by the Code4Community Club, this system makes it easy for students to request tutoring assistance and for tutors to manage their tutoring sessions efficiently.

KEY FEATURES
------------

1. DUAL-ROLE SYSTEM
   - Students can request tutoring for specific math courses
   - Tutors can view and accept tutoring requests
   - Role-based access control ensures proper permissions

2. COURSE-SPECIFIC TUTORING
   Supported courses include:
   • Algebra 1
   • Algebra 2
   • Algebra 2 Trig
   • Functions
   • Trig with Advanced Algebra
   • Geometry

3. REAL-TIME MATCHING SYSTEM
   - Students submit requests for specific courses
   - Tutors see pending requests in real-time
   - Instant notifications when tutors accept requests
   - Live session tracking with timers

4. SESSION MANAGEMENT
   - Complete session lifecycle tracking (pending → accepted → started → completed)
   - Real-time session timers for both tutors and students
   - Automatic session history recording
   - Session completion summaries

5. SECURE LOGIN SYSTEM
   - Students and tutors can log in using their Google accounts
   - Alternative email/password login option
   - Secure user verification process
   - Different access levels for students, teachers, and administrators

6. ADMINISTRATIVE FEATURES
   - Admin dashboard for overseeing the system
   - User management capabilities
   - Session history and reports
   - System monitoring tools

HOW THE SYSTEM WORKS
--------------------

STUDENT WORKFLOW:
1. Student logs in using Google or email/password
2. Selects their math course from the dropdown menu
3. Submits a tutoring request
4. System searches for available tutors
5. When matched, student sees tutor information and session status
6. During active sessions, real-time timer tracks duration
7. Session completion shows summary with duration and details

TUTOR WORKFLOW:
1. Tutor logs in and accesses tutor dashboard
2. Views pending student requests in real-time
3. Accepts requests for courses they can tutor
4. Starts tutoring session when ready
5. Monitors session with live timer
6. Ends session when tutoring is complete
7. Session data is automatically saved to history
